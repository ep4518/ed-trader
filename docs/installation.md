# Installation

## Stable release

To install ed_trader, run this command in your terminal:

```sh
uv add ed-trader
```

Or if you prefer to use `pip`:

```sh
pip install ed-trader
```

## From source

The source files for ed_trader can be downloaded from the [Github repo](https://github.com/ep4518/ed_trader).

You can either clone the public repository:

```sh
git clone https://github.com/ep4518/ed_trader
```

Or download the [tarball](https://github.com/ep4518/ed_trader/tarball/main):

```sh
curl -OJL https://github.com/ep4518/ed_trader/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd ed_trader
uv sync
```
